# Sample-data-analytics
Data analytics
run this file, you'll need Python installed, along with the pandas and matplotlib libraries.

Save the code: Save the content above into a file named expense_analysis.py.

Install dependencies: Open your terminal or command prompt and run:

Bash

pip install pandas matplotlib
Run the script: Execute the file from your terminal:

Bash

python expense_analysis.py
This will print the data and summary statistics to your console and display the three generated charts (Pie, Bar, and Line).
